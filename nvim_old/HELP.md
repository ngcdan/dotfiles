# search and replace 
command mode: 
```
:%s/search/replace/gc
```
- % : scope - current buffer
- g : all occurrences 
- c : ask for confirmation on each match

### Change working dir 
```
:b + Tab : switch to buffers
:bp : switch to previous buffer
```
